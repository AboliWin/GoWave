for linux: ```pip install python-vlc ```

for windows:
‍‍```pip install python-vlc windows-curses```

run:
```python3 radio.py```

- Play the song by pressing numbers and letters on the keyboard
- Space or "p" for play/pause
- UP & DOWN key for change volume
- "Esc" for exit
